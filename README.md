# Student Management System – University Group Project

## Overview
The Student Management System is a web-based academic project developed as part of our university coursework. This group project was designed to manage student-related information efficiently, including student records, academic data, and administrative operations. The project demonstrates collaborative software development, system design, and practical implementation of core programming concepts.

## Project Type
University Group Project  
Academic Web Application

## Objectives
- Manage student information in a structured and organized manner  
- Practice teamwork and collaborative development  
- Apply software engineering principles learned in coursework  
- Build a functional system using real-world academic requirements  

## Technologies Used
- HTML  
- CSS

## Key Features
- User authentication system  
- Student record creation and management  
- Database-driven dynamic content  
- Organized backend logic  
- Clean and structured UI  
- CRUD operations (Create, Read, Update, Delete)

## How to Run the Project
1. Install **XAMPP** or any local server environment  
2. Copy the project folder into the `htdocs` directory  
3. Start **Apache** and **MySQL** from XAMPP Control Panel  
4. Import the database:
   - Open `phpMyAdmin`
   - Create a database
